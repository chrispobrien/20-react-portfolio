# My react portfolio

## Description

A react-based portfolio of my work.

![screenshot]

## Installation

Download the project, install dependencies

```sh
git clone https://github.com/chrispobrien/20-react-portfolio.git
cd 20-react-portfolio
npm i
```

## Configuration

No need for configuration.

Start the server.

```sh
npm start
```
